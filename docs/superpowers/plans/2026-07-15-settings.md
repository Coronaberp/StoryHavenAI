# Settings (new_ui) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a full Settings feature for `new_ui/` — a grouped entry list plus three focused sub-screens (Appearance, Model & memory, Account) — matching `docs/superpowers/specs/2026-07-15-settings-design.md`.

**Architecture:** One `*View` class per screen (`SettingsView`, `AppearanceSettingsView`, `ModelSettingsView`, `AccountSettingsView`) in `new_ui/js/settings*.js`, each `mount(main)`-based like `PantheonView`/`ArtisanProfileView`, registered as flat routes in `new_ui/js/router.js`. `new_ui/js/theme.js` is extended from a flat preset-index cycler into a per-base (dark/light) state object so each mode keeps its own independent accent preset and optional custom-hex overrides. All Model/Account fields round-trip through the existing `GET/PUT/DELETE /api/me/settings`, `PUT /api/me/nsfw`, and `PUT /api/auth/password` endpoints exactly as `legacy_ui/js/user-settings.js` does today — no backend changes.

**Tech Stack:** Vanilla JS (no framework, no build step for JS), Tailwind CSS v4 compiled via `./rebuild.sh` into `new_ui/css/app.css`, served by `dev_server.py` on `:3001` proxying `/api/*` to the real backend on `:3000`.

## Global Constraints

- Never use `EnterWorktree`/`git worktree` for this repo — edit `/var/home/staygold/ai-frontend` directly (this checkout is the live bind mount / dev-server source).
- Zero comments in any file, ever — no exceptions (project-wide rule, `CLAUDE.md` "Coding style").
- Every `backend/routers/*`, `backend/repositories/*` file already exists and is untouched by this plan — this plan is frontend-only (`new_ui/`).
- `new_ui/` has no JS test/unit-test harness. In place of automated unit tests, every task's verification step is either (a) a `curl` check that the file is served correctly, or (b) a Playwright script driven against the already-running `:3001` dev server (never spin up a second instance — see `CLAUDE.md`'s "UI rework in progress" section).
- All new/edited CSS must go through the Tailwind build (`./bin/tailwindcss -i new_ui/css/input.css -o new_ui/css/app.css`, or rely on the human's running `./rebuild.sh --watch`) — never hand-edit `new_ui/css/app.css` directly, it's a build artifact.
- Any inline `style="..."` written by these views must reference the CSS custom properties (`var(--color-accent)`, etc.), never a hardcoded hex, per `CLAUDE.md`'s theme-system rule.
- Route names in this plan are flat (`settings`, `settings-appearance`, `settings-model`, `settings-account`), not nested paths — `router.js`'s `currentRoute()` only reads the first path segment (see `new_ui/js/router.js:82`), so nested paths like `/settings/appearance` would all resolve to the same route. Flat names match the existing convention (`pinacotheca`, `symposium`, etc. are all flat siblings of `compendium`, not nested under it).
- Dropped from scope entirely, matching the approved spec: Notifications preferences, Generation credits/billing, Creator themes toggle, ComfyUI backend status row, Admin panel row, Output gallery/LoRA rows (no `new_ui` routes exist for these yet).

### Task 1: Per-base independent theme state in `theme.js`

**Files:**
- Modify: `new_ui/js/theme.js` (full rewrite of its current 35 lines)

**Interfaces:**
- Consumes: `store.get`/`store.set` (`new_ui/js/state-store.js`, already loaded before `theme.js` in `index.html`).
- Produces (used by Tasks 3–5 and by the existing `index.html` temp-icon button `onclick="cycleTheme()"`):
  - `ACCENT_IDS: string[]`, `ACCENT_NAMES: {[id: string]: string}`, `ACCENT_SEED_HEX: {[id: string]: string}` — constants for building swatch pickers.
  - `getThemeState(): { activeBase: "dark"|"light", dark: {accentId: string, overrides: {accent?, text?, appBg?}}, light: {accentId: string, overrides: {...}} }`
  - `setThemeBase(base: "dark"|"light"): void` — switches which base is live.
  - `setThemeAccent(base: "dark"|"light", accentId: string): void` — picks a named preset for that base, clearing its overrides.
  - `setThemeOverride(base: "dark"|"light", key: "accent"|"text"|"appBg", hexValue: string): void` — sets one custom override for that base and marks its `accentId` as `"custom"`.
  - `clearThemeOverrides(base: "dark"|"light"): void` — drops that base's overrides and falls back to its last named preset (or `"aurum"` if it was `"custom"` with no prior preset).
  - `cycleTheme(): void` — kept for the existing temp icon button; now toggles `activeBase` only (dark↔light), each base keeping its own remembered accent.
  - `applyTheme(): void` — re-applies `THEME_STATE` to `<html>` and persists it; called internally by every setter above.

- [ ] **Step 1: Write the new `theme.js`**

```js
"use strict";

const THEME_BASES = ["dark", "light"];
const ACCENT_IDS = ["aurum", "azure", "crimson", "verdant", "amethyst", "rose"];
const ACCENT_NAMES = {
  aurum: "Aurum", azure: "Azure", crimson: "Crimson",
  verdant: "Verdant", amethyst: "Amethyst", rose: "Rose Quartz",
};
const ACCENT_SEED_HEX = {
  aurum: "#E3BD6C", azure: "#4FB6FF", crimson: "#E2493D",
  verdant: "#3ECF8E", amethyst: "#A675E0", rose: "#E07BA0",
};

const SUN_ICON = '<svg width="17" height="17" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><circle cx="12" cy="12" r="4.5"/><line x1="12" y1="1.5" x2="12" y2="4"/><line x1="12" y1="20" x2="12" y2="22.5"/><line x1="4.2" y1="4.2" x2="5.9" y2="5.9"/><line x1="18.1" y1="18.1" x2="19.8" y2="19.8"/><line x1="1.5" y1="12" x2="4" y2="12"/><line x1="20" y1="12" x2="22.5" y2="12"/><line x1="4.2" y1="19.8" x2="5.9" y2="18.1"/><line x1="18.1" y1="5.9" x2="19.8" y2="4.2"/></svg>';
const MOON_ICON = '<svg width="17" height="17" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M21 12.8A9 9 0 1 1 11.2 3a7 7 0 0 0 9.8 9.8z"/></svg>';

const OVERRIDE_CSS_VAR = { accent: "--color-accent", text: "--color-ink", appBg: "--color-paper" };

function _defaultThemeState() {
  return {
    activeBase: "dark",
    dark: { accentId: "aurum", overrides: {} },
    light: { accentId: "aurum", overrides: {} },
  };
}

function _loadThemeState() {
  const saved = store.get("themeState", null);
  if (saved && saved.dark && saved.light && saved.activeBase) return saved;
  const legacyIndex = store.get("themeIndex", null);
  if (legacyIndex !== null && Number.isInteger(legacyIndex)) {
    const perBase = ACCENT_IDS.length;
    const base = THEME_BASES[Math.floor(legacyIndex / perBase) % THEME_BASES.length];
    const accentId = ACCENT_IDS[legacyIndex % perBase] || "aurum";
    return { activeBase: base, dark: { accentId, overrides: {} }, light: { accentId, overrides: {} } };
  }
  return _defaultThemeState();
}

let THEME_STATE = _loadThemeState();

function getThemeState() {
  return THEME_STATE;
}

function applyTheme() {
  const root = document.documentElement;
  const base = THEME_STATE.activeBase;
  const baseState = THEME_STATE[base];
  root.dataset.theme = base;
  root.dataset.accent = baseState.accentId;
  Object.entries(OVERRIDE_CSS_VAR).forEach(([key, cssVar]) => {
    const value = baseState.overrides[key];
    if (value) root.style.setProperty(cssVar, value);
    else root.style.removeProperty(cssVar);
  });
  store.set("themeState", THEME_STATE);
  const icon = document.querySelector("[data-theme-icon]");
  if (icon) icon.innerHTML = base === "light" ? SUN_ICON : MOON_ICON;
}

function setThemeBase(base) {
  THEME_STATE.activeBase = base;
  applyTheme();
}

function setThemeAccent(base, accentId) {
  THEME_STATE[base] = { accentId, overrides: {} };
  applyTheme();
}

function setThemeOverride(base, key, hexValue) {
  THEME_STATE[base].accentId = "custom";
  THEME_STATE[base].overrides[key] = hexValue;
  applyTheme();
}

function clearThemeOverrides(base) {
  const fallbackAccent = THEME_STATE[base].accentId === "custom" ? "aurum" : THEME_STATE[base].accentId;
  THEME_STATE[base] = { accentId: fallbackAccent, overrides: {} };
  applyTheme();
}

function cycleTheme() {
  setThemeBase(THEME_STATE.activeBase === "dark" ? "light" : "dark");
}

applyTheme();
```

      ME.nsfw_allowed = true;
      toast("Mature content enabled.");
    } catch (e) { errorToast(e.message); }
    this.render();
  }

    cycleCensor();
    this.render();
  }

class AppearanceSettingsView {

  setOverride(key, hexValue) {

  { key: "narration", label: "Narration", defaultFlags: "i", sample: "The tavern door creaks open, spilling lamplight onto the cobblestones." },
  { key: "dialogue", label: "Dialogue", defaultFlags: "", sample: '"You\'re later than I expected," she says, not looking up.' },
  { key: "thoughts", label: "Thoughts", defaultFlags: "", sample: "This was a mistake. I should have brought the map." },
  { key: "voice", label: "Voice/OOC", defaultFlags: "ib", sample: "(Let's skip ahead to the morning after.)" },
  { key: "bold", label: "Bold", defaultFlags: "b", sample: "The floor gives way without warning." },
];
const MESSAGE_STYLE_DEFAULT_COLOR = "#E3BD6C";

function getMessageStyle() {
  const saved = store.get("messageStyle", null);
  if (saved) return saved;
  const fresh = {};
  MESSAGE_STYLE_CATEGORIES.forEach((cat) => {
    fresh[cat.key] = { font: "", color: MESSAGE_STYLE_DEFAULT_COLOR, flags: cat.defaultFlags };
  });
  return fresh;
}

function setMessageStyle(style) {
  store.set("messageStyle", style);
}

function specimenLineStyle(entry) {
  const parts = [`color:${entry.color || MESSAGE_STYLE_DEFAULT_COLOR}`];
  if (entry.font) parts.push(`font-family:'${entry.font}'`);
  if (entry.flags.includes("i")) parts.push("font-style:italic");
  if (entry.flags.includes("b")) parts.push("font-weight:700");
  if (entry.flags.includes("u")) parts.push("text-decoration:underline");
  if (entry.flags.includes("s")) parts.push(entry.flags.includes("u") ? "text-decoration:underline line-through" : "text-decoration:line-through");
  return parts.join(";");
}

        <textarea id="model_suffix" class="w-full px-2.5 py-2 rounded-md border border-line bg-surface text-ink text-sm" style="min-height:68px">${o.system_suffix || ""}</textarea>
      </div>
      <div class="mb-5">
        <label class="block text-xs text-sec mb-1">Post-history instructions</label>
        <textarea id="model_posthist" class="w-full px-2.5 py-2 rounded-md border border-line bg-surface text-ink text-sm" style="min-height:68px">${o.post_history || ""}</textarea>
      </div>

      <div class="flex gap-2">
        <button type="button" onclick="modelView.save()" ${this.saving ? "disabled" : ""}
          class="flex-1 py-3 rounded-xl font-semibold text-sm text-paper bg-gradient-to-br from-primary to-primary-dark disabled:opacity-60">
          ${this.saving ? "Saving…" : "Save changes"}
        </button>
        <button type="button" onclick="modelView.resetAll()" class="px-4 py-3 rounded-xl border text-sm" style="border-color:var(--color-warn);color:var(--color-warn)">Reset</button>
      </div>
    `;
  }

  async fetchModels() {
    const base = document.getElementById("model_base_url")?.value.trim() || this.settings.defaults?.base_url || "";
    const key = document.getElementById("model_api_key")?.value.trim() || "";
    const params = new URLSearchParams();
    if (base) params.set("base_url", base);
    if (key) params.set("api_key", key);
    try {
      const { models } = await api("/api/models" + (params.toString() ? "?" + params : ""));
      const list = document.getElementById("model_model_list");
      if (!models?.length) { toast("No models returned"); return; }
      list.innerHTML = models.map((m) => `<button type="button" class="px-2 py-1 rounded-md border border-line bg-surface-2 text-xs" onclick="document.getElementById('model_chat_model').value='${m.replace(/'/g, "\\'")}'">${m}</button>`).join("");
    } catch (e) {
      errorToast("Fetch failed: " + e.message);
    }
  }

  numOrNull(id) {
    const v = parseFloat(document.getElementById(id)?.value ?? "");
    return isNaN(v) ? null : v;
  }

  intOrNull(id) {
    const v = parseInt(document.getElementById(id)?.value ?? "", 10);
    return isNaN(v) ? null : v;
  }

  async save() {
    this.saving = true;
    this.render();
    const body = {
      history_turns: this.intOrNull("model_history_turns"),
      max_tokens: this.intOrNull("model_max_tokens"),
      enable_thinking: !!document.getElementById("model_thinking")?.checked,
      scene_style: !!document.getElementById("model_scene")?.checked,
      temperature: this.numOrNull("model_temperature"),
      top_p: this.numOrNull("model_top_p"),
      top_k: this.intOrNull("model_top_k"),
      min_p: this.numOrNull("model_min_p"),
      top_a: this.numOrNull("model_top_a"),
      typical_p: this.numOrNull("model_typical_p"),
      repetition_penalty: this.numOrNull("model_repetition_penalty"),
      frequency_penalty: this.numOrNull("model_frequency_penalty"),
      presence_penalty: this.numOrNull("model_presence_penalty"),
      seed: this.intOrNull("model_seed"),
      stop: (document.getElementById("model_stop")?.value || "").split("\n").map((s) => s.trim()).filter(Boolean),
      system_suffix: document.getElementById("model_suffix")?.value.trim() || null,
      post_history: document.getElementById("model_posthist")?.value.trim() || null,
    };
    body.stop = body.stop.length ? body.stop : null;
    if (document.getElementById("model_use_own")?.checked) {
      body.base_url = document.getElementById("model_base_url")?.value.trim() || null;
      body.chat_model = document.getElementById("model_chat_model")?.value.trim() || null;
      const key = document.getElementById("model_api_key")?.value;
      if (key) body.api_key = key;
    } else {
      body.base_url = null;
      body.chat_model = null;
      body.api_key = null;
    }
    try {
      await api("/api/me/settings", { method: "PUT", body: JSON.stringify(body) });
      this.settings = await api("/api/me/settings");
      toast("Settings saved.");
    } catch (e) {
      errorToast("Save failed: " + e.message);
    }
    this.saving = false;
    this.render();
  }

  async resetAll() {
    if (!confirm("Reset all model & memory settings to defaults?")) return;
    try {
      await api("/api/me/settings", { method: "DELETE" });
      this.settings = await api("/api/me/settings");
      toast("Reset to defaults.");
    } catch (e) {
      errorToast("Reset failed: " + e.message);
    }
    this.render();
  }
}

if (typeof window !== "undefined") {

}

class AccountSettingsView {
  async mount(main) {

        <label class="block text-xs text-sec mb-1">Current password</label>

      <button type="button" onclick="accountView.changePassword()" class="w-full mb-6 py-2.5 rounded-xl font-semibold text-sm text-paper bg-gradient-to-br from-primary to-primary-dark">
        Change password
      </button>

      ${sEyebrowHtml("Language")}
      <div class="mb-2">
        <label class="block text-xs text-sec mb-1">Interface language</label>
        <input type="text" id="acct_lang" value="${lang}" placeholder="English" class="w-full px-2.5 py-2 rounded-md border border-line bg-surface text-ink text-sm">
      </div>
      <button type="button" onclick="accountView.saveLanguage()" class="w-full py-2.5 rounded-xl border text-sm" style="border-color:var(--color-line);color:var(--color-ink)">
        Save language
      </button>
    `;
  }

  async changePassword() {
    const oldPw = document.getElementById("acct_old_pw").value;
    const newPw = document.getElementById("acct_new_pw").value;
    const confirmPw = document.getElementById("acct_confirm_pw").value;
    if (!oldPw || !newPw) return;
    if (newPw !== confirmPw) { errorToast("New passwords don't match."); return; }
    try {
      await api("/api/auth/password", { method: "PUT", body: JSON.stringify({ old_password: oldPw, new_password: newPw }) });
      toast("Password changed.");
      document.getElementById("acct_old_pw").value = "";
      document.getElementById("acct_new_pw").value = "";
      document.getElementById("acct_confirm_pw").value = "";
    } catch (e) {
      errorToast(e.message);
    }
  }

  async saveLanguage() {
    const lang = document.getElementById("acct_lang").value.trim();
    try {
      await api("/api/me/settings", { method: "PUT", body: JSON.stringify({ interface_language: lang || null }) });
      toast("Language saved. Reloading…");
      setTimeout(() => location.reload(), 600);
    } catch (e) {
      errorToast("Save failed: " + e.message);
    }
  }
}
